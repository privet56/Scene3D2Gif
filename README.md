# Scene3D2Gif

## Pre-Requisites:
	Visual Studio 2017 / Community Edition with Desktop WPF Support
	.NET Framework 4.7.2 Developer Pack

## NuGet Packages:
	Install-Package HelixToolkit, HelixToolkit.Wpf, HelixToolkit.Wpf.SharpDX
